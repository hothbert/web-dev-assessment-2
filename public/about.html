Supplimentary Information

What is RCS?

The Royal Conservatoire of Scotland is a conservatoire of dance, music, drama, film and production, in Glasgow, Scotland. I studied in the Junior Conservatoire of Music, where I did classical piano and flute. The Junior Conservatoire is a program for school age children.

What is ABRSM?

The Assocciated Board of the Royal Schools of Music is an examination board based in the UK.

Why take a course with the Open University?

My school did not offer Advanced Higher Computing Science as a subject so I decided to do this course instead so that I could continue my learning.

Development Process

The website development process started by making a framework/general layout, to be kept consistent across the web pages (although it was simplified for the chat page). Bootstrap and styles were added to improve the appearence of pages. Content was then added to both the home and about pages, as well as some additional features such as a card, accordion and a carousel of images. Next, all HTML and CSS code was validated to check for errors. Lastly, the chat application was created. Git commits were made throughout the proccess.
